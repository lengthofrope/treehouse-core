Class PluginLoadPriority

All Treehouse plugins and themes depend on the Treehouse CORE plugin since this plugin houses all shared code. This class makes sure we load the Treehouse CORE plugin before any other Treehouse plugin without the need of converting to a mu-plugin since they are less flexible and can only be manually updated.

Note: themes are loaded at a later stage by default.
