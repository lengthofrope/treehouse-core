  5: /**
  6:  * A helper class to create Custom Post Types.
  7:  *
  8:  * It handles:
  9:  * - Creation of the post type (with a prefix by default)
 10:  * - Adding templates within the plugin for singles and archives (using setupTemplates method).
 11:  * - Allows basic settings to be set through chaining
 12:  *
 13:  * Note: To make sure all translations are ready this should be called in an init action!
 14:  *
 15:  * Usage (ordinary mode):
 16:  * <code>
 17:  * new \LengthOfRope\Treehouse\CPT\Create('products', array(
 18:  *      'labels'      => array(
 19:  *          'name'          => __('Products'),
 20:  *          'singular_name' => __('Product')

 22:  *      'public'      => true,
 23:  *      'has_archive' => true,

 28:  * <code>
 29:  * \LengthOfRope\Treehouse\CPT\Create::factory('products')

 31:  *      ->setPublic(true)
 32:  *      ->setHasArchive(true);
